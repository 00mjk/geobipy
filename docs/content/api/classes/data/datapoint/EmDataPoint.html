find_best_halfspace(minConductivity=0.0001, maxConductivity=10000.0, nSamples=100)ÔÉÅ

Computes the best value of a half space that fits the data.

Carries out a brute force search of the halfspace conductivity that best fits the data. The profile of data misfit vs halfspace conductivity is not quadratic, so a bisection will not work.

Parameters

  • minConductivity (float, optional) ‚Äì The minimum conductivity to search over

  • maxConductivity (float, optional) ‚Äì The maximum conductivity to search over

  • nSamples (int, optional) ‚Äì The number of values between the min and max

Returns

out ‚Äì The best fitting log10 conductivity for the half space

Return type
