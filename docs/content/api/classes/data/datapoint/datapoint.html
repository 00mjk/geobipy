Class defines a data point.

Contains an easting, northing, height, elevation, observed and predicted data, and uncertainty estimates for the data.

DataPoint(x, y, z, elevation, nChannels, data, std, units)

Parameters

  • nChannelsPerSystem (int or array_like) ‚Äì Number of data channels in the data * If int, a single acquisition system is assumed. * If array_like, each entry is the number of channels for each system.

  • x (float) ‚Äì Easting co-ordinate of the data point

  • y (float) ‚Äì Northing co-ordinate of the data point

  • z (float) ‚Äì Height above ground of the data point

  • elevation (float, optional) ‚Äì Elevation from sea level of the data point

  • data (geobipy.StatArray or array_like, optional) ‚Äì Data values to assign the data of length sum(nChannelsPerSystem). * If None, initialized with zeros.

  • std (geobipy.StatArray or array_like, optional) ‚Äì Estimated uncertainty standard deviation of the data of length sum(nChannelsPerSystem). * If None, initialized with ones if data is None, else 0.1*data values.

  • predictedData (geobipy.StatArray or array_like, optional) ‚Äì Predicted data values to assign the data of length sum(nChannelsPerSystem). * If None, initialized with zeros.

  • units (str, optional) ‚Äì Units of the data. Default is ‚Äúppm‚Äù.

  • channelNames (list of str, optional) ‚Äì Names of each channel of length sum(nChannelsPerSystem)

updateErrors(relativeErr, additiveErr)ÔÉÅ

Updates the data errors.

Updates the standard deviation of the data errors using the following model

\[\sqrt{(\mathbf{\epsilon}_{rel} \mathbf{d}^{obs})^{2} + \mathbf{\epsilon}^{2}_{add}},\]

where \(\mathbf{\epsilon}_{rel}\) is the relative error, a percentage fraction and \(\mathbf{\epsilon}_{add}\) is the additive error.

If the predicted data have been assigned a multivariate normal distribution, the variance of that distribution is also updated as the squared standard deviations.

Parameters

  • relativeErr (float or array_like) ‚Äì A fraction percentage that is multiplied by the observed data.

  • additiveErr (float or array_like) ‚Äì An absolute value of additive error.

Raises

  • ValueError ‚Äì If relativeError is <= 0.0

  • ValueError ‚Äì If additiveError is <= 0.0
